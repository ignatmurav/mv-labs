## 2. Постановка задачи  

На отрезке $\([a, b] = [-2, 2]\)$ для двух функций  

$f_1(x) = \sin(2x)\ln(x+5)$,
$f_2(x) = \sqrt{2|x| + x^2}$

требуется построить интерполяционный кубический сплайн $\(S_3(x)\)$ с граничными условиями первого рода (зажатый сплайн):  
$\[
S_3'(a) = f'(a), \quad S_3'(b) = f'(b).
\]$ 
Узлы интерполяции выбираются равноотстоящими:  
$\[
x_i = a + ih,\quad i = 0,\dots,N,\quad h = \frac{b-a}{N},
\]$  
где $\(N\)$ – число разбиений отрезка.  

Программа должна:  

1. Для $\(N = 15\)$ построить совмещённый график $\(f(x)\)$ и сплайна $\(S_3(x)\)$, а также вычислить норму погрешности  
$\[
\|r_N\| = \max_{i=0,\dots,500} |f(\bar{x}_i) - S_3(\bar{x}_i)|,
\]$  
где $\(\bar{x}_i = a + i\frac{b-a}{500}\)$.  

2. Построить график сходимости в логарифмическом масштабе: по оси $\(Ox\)$ – количество разбиений $\(N\)$ от 5 до 100 с шагом 5, по оси $\(Oy\)$ – равномерная норма погрешности $\(\|r_N\|\)$.  

3. Сделать выводы о сходимости интерполяционного кубического сплайна для различных функций.  

Решение системы с трёхдиагональной матрицей должно выполняться методом прогонки.

## 3. Теоретические сведения  

### 3.1. Построение кубического сплайна  

Кубическим сплайном называется функция $\(S_3(x)\)$, которая на каждом интервале $\([x_i, x_{i+1}]\)$ является многочленом третьей степени и удовлетворяет условиям непрерывности самой функции, её первой и второй производных во всех внутренних узлах.  

Для нахождения сплайна используют представление на \(i\)-м интервале:  
$\[
S_3(x) = a_i + b_i (x - x_i) + c_i (x - x_i)^2 + d_i (x - x_i)^3, \quad x \in [x_i, x_{i+1}].
\]$

Из условий интерполяции $\(S_3(x_i) = y_i\) и \(S_3(x_{i+1}) = y_{i+1}\)$ получаем:  
$\[
a_i = y_i, \quad a_i + b_i h + c_i h^2 + d_i h^3 = y_{i+1}.
\]$ 

Из непрерывности первой производной:  
$\[
b_i + 2c_i h + 3d_i h^2 = b_{i+1}.
\]$  

Из непрерывности второй производной:  
$\[
2c_i + 6d_i h = 2c_{i+1}.
\]$  

Вводя вторые производные $\(m_i = S_3''(x_i) = 2c_i\)$, получаем систему для $\(m_i\)$:  

$\[
h m_{i-1} + 4h m_i + h m_{i+1} = 6\left(\frac{y_{i+1} - y_i}{h} - \frac{y_i - y_{i-1}}{h}\right),\quad i = 1,2,\dots,N-1.
\]$  

Для граничных условий первого рода (зажатый сплайн) заданы значения первых производных на концах:  
$\(S_3'(a) = y_0'\), \(S_3'(b) = y_N'\)$. Это даёт уравнения:  

$\[
2h m_0 + h m_1 = 6\left(\frac{y_1 - y_0}{h} - y_0'\right),
\]$
$\[
h m_{N-1} + 2h m_N = 6\left(y_N' - \frac{y_N - y_{N-1}}{h}\right).
\]$  

Таким образом, получаем трёхдиагональную систему относительно $\(m_i\) (\(i = 0,\dots,N\))$:  

$\[
\begin{cases}
$2 m_0 + 1 \cdot m_1 = d_0,\\$
1\cdot m_{i-1} + 4 m_i + 1\cdot m_{i+1} = d_i,$\quad i=1,\dots,N-1,\\$
1\cdot m_{N-1} + 2 m_N = d_N,$
$\end{cases}
\]$

где  
$\[
d_0 = \frac{6}{h^2}(y_1 - y_0) - \frac{6}{h}y_0',\quad
d_i = \frac{6}{h^2}(y_{i+1} - 2y_i + y_{i-1}),\quad
d_N = \frac{6}{h}y_N' - \frac{6}{h^2}(y_N - y_{N-1}).
\]$ 

### 3.2. Метод прогонки  

Система с трёхдиагональной матрицей решается методом прогонки (алгоритм Томаса).  
Прямой ход:  
$\[
\alpha_0 = \frac{c_0}{b_0},\quad \beta_0 = \frac{d_0}{b_0},
\]$  
$\[
\alpha_i = \frac{c_i}{b_i - a_i\alpha_{i-1}},\quad
\beta_i = \frac{d_i - a_i\beta_{i-1}}{b_i - a_i\alpha_{i-1}},\quad i = 1,\dots,N-1.
\]$  

Обратный ход:  
$\[
m_N = \beta_N,\quad m_i = \beta_i - \alpha_i m_{i+1},\quad i = N-1,\dots,0.
\]$

После нахождения $\(m_i\)$ коэффициенты сплайна вычисляются по формулам:  
$\[
c_i = \frac{m_i}{2},\quad d_i = \frac{m_{i+1} - m_i}{6h},
\]$ 
$\[
b_i = \frac{y_{i+1} - y_i}{h} - \frac{h}{6}(2m_i + m_{i+1}).
\]$

### 4.3. Анализ сходимости  

- Для **гладкой функции** $\(f_1(x)\)$ погрешность убывает очень быстро: при увеличении $\(N\)$ в 20 раз (с 5 до 100) погрешность уменьшается более чем в $\(10^4\)$ раз. На логарифмическом графике зависимость близка к линейной, что соответствует теоретической сходимости $\(O(h^4)\)$ для кубического сплайна на гладких функциях.  
- Для **функции с изломом** $\(f_2(x)\)$ (производная терпит разрыв в нуле) сходимость значительно медленнее. Даже при $\(N=100\)$ погрешность составляет $\(\approx 1.7\times 10^{-3}\)$, что на три порядка больше, чем для гладкой функции при том же $\(N\)$. Это объясняется тем, что сплайн не может точно воспроизвести излом, и порядок сходимости снижается до $\(O(h^2)\)$ (или даже $\(O(h)\)$ в зависимости от гладкости).  

Таким образом, интерполяционный кубический сплайн эффективно аппроксимирует гладкие функции, но для функций с особенностями (разрывы производных) сходимость замедляется.

## 5. Листинг программы с комментариями  

```python
import numpy as np
import matplotlib.pyplot as plt

# ----------------------------------------------------------------------
# Задание функций и их производных
# ----------------------------------------------------------------------

def f1(x):
    """f1(x) = sin(2x) * ln(x+5)"""
    return np.sin(2*x) * np.log(x + 5)

def f1_prime(x):
    # Производная f1(x)
    return 2*np.cos(2*x)*np.log(x+5) + np.sin(2*x)/(x+5)

def f2(x):
    """f2(x) = sqrt(2|x| + x^2)"""
    return np.sqrt(2*np.abs(x) + x**2)

def f2_prime(x):
    # Производная f2(x) (на концах отрезка конечна)
    if x < 0:
        return (x - 1) / np.sqrt(x**2 - 2*x)
    elif x > 0:
        return (x + 1) / np.sqrt(x**2 + 2*x)
    else:
        # Производная в нуле бесконечна, но в узлах сетки её нет
        return np.inf

# ----------------------------------------------------------------------
# Построение кубического сплайна с зажатыми граничными условиями
# ----------------------------------------------------------------------

def cubic_spline_coeffs(x, y, yp_left, yp_right):
    """
    Вычисление коэффициентов кубического сплайна на равномерной сетке.

    Параметры:
        x : массив узлов (должны быть равноотстоящими)
        y : значения функции в узлах
        yp_left : первая производная в левом конце
        yp_right : первая производная в правом конце

    Возвращает:
        coeffs : список кортежей (a_i, b_i, c_i, d_i) для каждого интервала
        x_nodes : входные узлы (те же, что и x)
    """
    n = len(x) - 1          # количество интервалов
    h = x[1] - x[0]         # шаг сетки (предполагается постоянным)

    # Вектор правой части d для вторых производных m_i
    d = np.zeros(n + 1)
    d[0] = 6/h**2 * (y[1] - y[0]) - 6/h * yp_left
    for i in range(1, n):
        d[i] = 6/h**2 * (y[i+1] - 2*y[i] + y[i-1])
    d[n] = 6/h * yp_right - 6/h**2 * (y[n] - y[n-1])

    # Коэффициенты трёхдиагональной матрицы (a – нижняя диагональ,
    # b – главная, c – верхняя)
    a = np.zeros(n + 1)
    b = np.zeros(n + 1)
    c = np.zeros(n + 1)

    b[0] = 2.0
    c[0] = 1.0
    for i in range(1, n):
        a[i] = 1.0
        b[i] = 4.0
        c[i] = 1.0
    a[n] = 1.0
    b[n] = 2.0

    # Прямой ход метода прогонки (алгоритм Томаса)
    for i in range(1, n + 1):
        w = a[i] / b[i-1]
        b[i] -= w * c[i-1]
        d[i] -= w * d[i-1]

    # Обратный ход
    m = np.zeros(n + 1)
    m[n] = d[n] / b[n]
    for i in range(n - 1, -1, -1):
        m[i] = (d[i] - c[i] * m[i+1]) / b[i]

    # Вычисление коэффициентов полиномов на каждом интервале
    coeffs = []
    for i in range(n):
        a_i = y[i]
        c_i = m[i] / 2.0
        d_i = (m[i+1] - m[i]) / (6.0 * h)
        b_i = (y[i+1] - y[i]) / h - h * (2*m[i] + m[i+1]) / 6.0
        coeffs.append((a_i, b_i, c_i, d_i))

    return coeffs, x

# ----------------------------------------------------------------------
# Вычисление сплайна в произвольных точках
# ----------------------------------------------------------------------

def eval_spline(coeffs, x_nodes, x_eval):
    """
    Вычисление значения сплайна в заданных точках.

    Параметры:
        coeffs : список коэффициентов по интервалам (из cubic_spline_coeffs)
        x_nodes : узлы сплайна
        x_eval : скаляр или массив точек, в которых вычисляется сплайн

    Возвращает:
        значения сплайна в точках x_eval
    """
    n = len(x_nodes) - 1
    if np.isscalar(x_eval):
        # Определение интервала
        if x_eval <= x_nodes[0]:
            i = 0
        elif x_eval >= x_nodes[-1]:
            i = n - 1
        else:
            i = np.searchsorted(x_nodes, x_eval) - 1
        a, b, c, d = coeffs[i]
        dx = x_eval - x_nodes[i]
        return a + b*dx + c*dx**2 + d*dx**3
    else:
        result = np.empty_like(x_eval)
        for idx, xv in enumerate(x_eval):
            if xv <= x_nodes[0]:
                i = 0
            elif xv >= x_nodes[-1]:
                i = n - 1
            else:
                i = np.searchsorted(x_nodes, xv) - 1
            a, b, c, d = coeffs[i]
            dx = xv - x_nodes[i]
            result[idx] = a + b*dx + c*dx**2 + d*dx**3
        return result

# ----------------------------------------------------------------------
# Основная часть вычислительного эксперимента
# ----------------------------------------------------------------------

a, b = -2.0, 2.0
funcs = [(f1, f1_prime, "f1(x) = sin(2x)·ln(x+5)"),
         (f2, f2_prime, "f2(x) = sqrt(2|x| + x^2)")]

for f, f_prime, title in funcs:
    print(f"\n=== {title} ===")

    # ------------------------------------------------------------------
    # Эксперимент 1: N = 15, построить графики функции и сплайна, вывести погрешность
    # ------------------------------------------------------------------
    N = 15
    x_nodes = np.linspace(a, b, N+1)
    y_nodes = f(x_nodes)
    yp_left = f_prime(a)
    yp_right = f_prime(b)

    coeffs, nodes = cubic_spline_coeffs(x_nodes, y_nodes, yp_left, yp_right)

    x_fine = np.linspace(a, b, 501)   # 500 интервалов → 501 точка
    y_fine = f(x_fine)
    y_spline = eval_spline(coeffs, nodes, x_fine)
    err = np.max(np.abs(y_fine - y_spline))
    print(f"N = {N}, максимальная погрешность = {err:.2e}")

    plt.figure()
    plt.plot(x_fine, y_fine, label='f(x)')
    plt.plot(x_fine, y_spline, '--', label='Сплайн')
    plt.title(f'{title}, N = {N}')
    plt.legend()
    plt.grid(True)
    plt.savefig(f'{f.__name__}_N15.png')
    plt.show()

    # ------------------------------------------------------------------
    # Эксперимент 2: график сходимости (N = 5,10,...,100)
    # ------------------------------------------------------------------
    N_list = list(range(5, 101, 5))
    errors = []
    for N in N_list:
        x_nodes = np.linspace(a, b, N+1)
        y_nodes = f(x_nodes)
        yp_left = f_prime(a)
        yp_right = f_prime(b)
        coeffs, nodes = cubic_spline_coeffs(x_nodes, y_nodes, yp_left, yp_right)
        x_fine = np.linspace(a, b, 501)
        y_fine = f(x_fine)
        y_spline = eval_spline(coeffs, nodes, x_fine)
        err = np.max(np.abs(y_fine - y_spline))
        errors.append(err)

    plt.figure()
    plt.loglog(N_list, errors, 'o-')
    plt.xlabel('Число интервалов N')
    plt.ylabel('Максимальная погрешность')
    plt.title(f'Сходимость: {title}')
    plt.grid(True, which='both')
    plt.savefig(f'{f.__name__}_convergence.png')
    plt.show()

    # Вывод таблицы значений
    print(f"N\tПогрешность")
    for N, err in zip(N_list, errors):
        print(f"{N}\t{err:.2e}")

print("\nВсе эксперименты завершены.")
```
