## Краткие теоретические сведения  

Интерполяция — способ приближения функции $f(x)$ многочленом $P_n(x)$ степени $n$, принимающим в заданных узлах $x_0, x_1, \dots, x_n$ те же значения, что и исходная функция.  

### Интерполяционный многочлен Ньютона  

Для неравноотстоящих узлов используется форма Ньютона с разделёнными разностями:

$`P_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \dots + f[x_0,\dots,x_n](x-x_0)\cdots(x-x_{n-1}),`$

где $f[x_i, x_{i+1}, \dots, x_{i+k}]$ — разделённые разности порядка $k$, вычисляемые рекуррентно:

$$
f[x_i] = f(x_i)
$$

$$
f[x_i,x_{i+1}] =
\frac{f(x_{i+1})-f(x_i)}{x_{i+1}-x_i}
$$

$$
f[x_i,\dots,x_{i+k}] =
\frac{f[x_{i+1},\dots,x_{i+k}] - f[x_i,\dots,x_{i+k-1}]}
{x_{i+k}-x_i}
$$

### Выбор узлов интерполяции  

В работе используются два типа узлов на отрезке $[a,b]$:

1. **Равноотстоящие узлы**

$$
x_i = a + i\frac{b-a}{n}, \quad i = 0,\dots,n
$$

2. **Чебышевские узлы (корни многочлена Чебышева)**

$$
x_i =
\frac{a+b}{2} +
\frac{b-a}{2}
\cos\left(
\frac{2i+1}{2(n+1)}\pi
\right),
\quad i = 0,\dots,n
$$

Они сгущаются к краям отрезка, что уменьшает погрешность интерполяции (борьба с явлением Рунге).

# Алгоритм вычисления значения интерполяционного многочлена Ньютона

Для эффективного вычисления $P_n(x)$ используется схема Горнера.

$$
P_n(x) =
f[x_0] +
(x-x_0)
\Big(
f[x_0,x_1] +
(x-x_1)
\big(
f[x_0,x_1,x_2] + \dots +
(x-x_{n-1})f[x_0,\dots,x_n]
\big)
\Big)
$$

Алгоритм:

1. Входные данные: массив узлов $x_0,\dots,x_n$, массив разделённых разностей $dd$.
2. Положить  $res = dd[n]$
3. Для $k = n-1, n-2, \dots, 0$;   $`res = res * (x - x_k) + dd[k]`$

4. Результат $res$ — значение $P_n(x)$.

## Листинг программы

Программа реализована на языке Python с использованием библиотек NumPy и Matplotlib.  

```python
import numpy as np
import matplotlib.pyplot as plt
import os

# Функции варианта 8
def f1(x):
    return np.sin(2*x) * np.log(x + 5)

def f2(x):
    return np.sqrt(2*np.abs(x) + x**2)

a, b = -2, 2

def equidistant_nodes(n):
    return np.linspace(a, b, n+1)

def chebyshev_nodes(n):
    i = np.arange(n+1)
    return (a+b)/2 + (b-a)/2 * np.cos((2*i+1)*np.pi/(2*(n+1)))

def divided_differences(x_nodes, y_nodes):
    """Вычисление разделённых разностей (одномерный массив, перезапись снизу вверх)"""
    n = len(x_nodes) - 1
    dd = np.copy(y_nodes).astype(float)
    for j in range(1, n+1):
        for i in range(n, j-1, -1):
            dd[i] = (dd[i] - dd[i-1]) / (x_nodes[i] - x_nodes[i-j])
    return dd

def newton_eval(x_nodes, dd, x):
    """Схема Горнера для многочлена Ньютона"""
    n = len(dd) - 1
    res = dd[n]
    for k in range(n-1, -1, -1):
        res = res * (x - x_nodes[k]) + dd[k]
    return res
```

## Выводы

В ходе лабораторной работы были реализованы алгоритмы построения интерполяционного многочлена Ньютона по равноотстоящим и чебышевским узлам, а также проведён вычислительный эксперимент для двух функций: гладкой $f_1(x) = \sin(2x)\ln(x+5)$ и функции с изломом $f_2(x) = \sqrt{2|x| + x^2}$.

1. Для гладкой функции $f_1$ интерполяционный процесс сходится при обоих типах узлов. Погрешность монотонно убывает с ростом степени многочлена. Чебышевские узлы дают несколько меньшую ошибку при одинаковых n, что объясняется их оптимальным распределением, уменьшающим константу в оценке погрешности.

2. Для функции с изломом $`f_2`$ наблюдается качественное различие:
   - При использовании равноотстоящих узлов после некоторой степени $(n ≈ 15)$ погрешность начинает расти из-за явления Рунге — осцилляций многочлена вблизи точки негладкости.
   - Чебышевские узлы существенно подавляют осцилляции; ошибка сначала убывает, а затем выходит на плато, ограниченное гладкостью функции.

3. Общий вывод: выбор узлов интерполяции критически влияет на сходимость. Для функций с ограниченной гладкостью (или разрывах производных) предпочтительнее использовать чебышевские узлы, которые минимизируют максимальную погрешность. Для аналитических функций оба подхода работоспособны, но чебышевские узлы дают более быструю сходимость.

Таким образом, интерполяционный процесс по чебышевским узлам является более устойчивым и универсальным, особенно при интерполяции функций с особенностями.
