## Краткие теоретические сведения  

Интерполяция — способ приближения функции \( f(x) \) многочленом \( P_n(x) \) степени \( n \), принимающим в заданных узлах \( x_0, x_1, \dots, x_n \) те же значения, что и исходная функция.  

### Интерполяционный многочлен Ньютона  
Для неравноотстоящих узлов используется форма Ньютона с разделёнными разностями:  

\[
P_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \dots + f[x_0,\dots,x_n](x-x_0)\cdots(x-x_{n-1}),
\]  

где \( f[x_i, x_{i+1}, \dots, x_{i+k}] \) — разделённые разности порядка \( k \), вычисляемые рекуррентно:  

\[
f[x_i] = f(x_i),\quad 
f[x_i,x_{i+1}] = \frac{f(x_{i+1})-f(x_i)}{x_{i+1}-x_i},\quad
f[x_i,\dots,x_{i+k}] = \frac{f[x_{i+1},\dots,x_{i+k}] - f[x_i,\dots,x_{i+k-1}]}{x_{i+k}-x_i}.
\]

### Выбор узлов интерполяции  
В работе используются два типа узлов на отрезке \([a,b]\):  

1. **Равноотстоящие узлы:**  
   \[
   x_i = a + i\frac{b-a}{n},\quad i = 0,\dots,n.
   \]  

2. **Чебышевские узлы (корни многочлена Чебышева):**  
   \[
   x_i = \frac{a+b}{2} + \frac{b-a}{2}\cos\left(\frac{2i+1}{2(n+1)}\pi\right),\quad i = 0,\dots,n.
   \]  
   Они сгущаются к краям отрезка, что уменьшает погрешность интерполяции для многих классов функций (борьба с явлением Рунге).

## Алгоритм вычисления значения интерполяционного многочлена Ньютона в заданной точке с применением схемы Горнера  

Для эффективного вычисления \( P_n(x) \) используется схема Горнера, аналогичная той, что применяется для многочленов в канонической форме. Представим многочлен Ньютона в виде вложенных произведений:  

\[
P_n(x) = f[x_0] + (x-x_0)\Big(f[x_0,x_1] + (x-x_1)\big(f[x_0,x_1,x_2] + \dots + (x-x_{n-1})f[x_0,\dots,x_n]\big)\Big).
\]

Тогда значение вычисляется последовательно, начиная с внутренней скобки:  

1. Входные данные: массив узлов \( x_0,\dots,x_n \), массив разделённых разностей \( dd \) (где \( dd[k] = f[x_0,\dots,x_k] \)), точка \( x \).  
2. Положить \( res = dd[n] \).  
3. Для \( k = n-1, n-2, \dots, 0 \):  
   \( res = res \cdot (x - x_k) + dd[k] \).  
4. Результат \( res \) — значение \( P_n(x) \).

**Вычисление разделённых разностей** выполняется «на месте» (in-place) в одномерном массиве, без построения полной таблицы:  

- Массив \( dd \) инициализируется значениями \( f(x_i) \).  
- Для \( j = 1 \) до \( n \):  
  для \( i = n \) вниз до \( j \):  
  \( dd[i] = (dd[i] - dd[i-1]) / (x_i - x_{i-j}) \).  
- По окончании \( dd[k] \) содержит \( f[x_0,\dots,x_k] \).

Такой подход экономит память и соответствует идее «записи значений в одномерный массив снизу вверх».

## Листинг программы  

Программа реализована на языке Python с использованием библиотек NumPy и Matplotlib.  

```python
import numpy as np
import matplotlib.pyplot as plt
import os

# Функции варианта 8
def f1(x):
    return np.sin(2*x) * np.log(x + 5)

def f2(x):
    return np.sqrt(2*np.abs(x) + x**2)

a, b = -2, 2

def equidistant_nodes(n):
    return np.linspace(a, b, n+1)

def chebyshev_nodes(n):
    i = np.arange(n+1)
    return (a+b)/2 + (b-a)/2 * np.cos((2*i+1)*np.pi/(2*(n+1)))

def divided_differences(x_nodes, y_nodes):
    """Вычисление разделённых разностей (одномерный массив, перезапись снизу вверх)"""
    n = len(x_nodes) - 1
    dd = np.copy(y_nodes).astype(float)
    for j in range(1, n+1):
        for i in range(n, j-1, -1):
            dd[i] = (dd[i] - dd[i-1]) / (x_nodes[i] - x_nodes[i-j])
    return dd

def newton_eval(x_nodes, dd, x):
    """Схема Горнера для многочлена Ньютона"""
    n = len(dd) - 1
    res = dd[n]
    for k in range(n-1, -1, -1):
        res = res * (x - x_nodes[k]) + dd[k]
    return res

def generate_plot_points(nodes_func, f, n):
    x_nodes = nodes_func(n)
    y_nodes = f(x_nodes)
    dd = divided_differences(x_nodes, y_nodes)
    x_grid = np.linspace(a, b, 501)
    y_interp = np.array([newton_eval(x_nodes, dd, xi) for xi in x_grid])
    return x_grid, y_interp

def save_points(filename, x_grid, y_interp):
    with open(filename, 'w') as f:
        for x, y in zip(x_grid, y_interp):
            f.write(f"{x:.6f} {y:.6f}\n")

def plot_interpolation(f, f_name, nodes_type, nodes_func):
    """Построение трёх графиков для n=2,10,20 и сохранение точек"""
    fig, axes = plt.subplots(1, 3, figsize=(15,5))
    fig.suptitle(f"{f_name}, узлы: {nodes_type}")
    for idx, n in enumerate([2,10,20]):
        ax = axes[idx]
        x_plot = np.linspace(a,b,1000)
        y_true = f(x_plot)
        ax.plot(x_plot, y_true, 'k-', label='f(x)', linewidth=2)
        x_grid, y_interp = generate_plot_points(nodes_func, f, n)
        ax.plot(x_grid, y_interp, 'r--', label=f'P_{n}(x)', linewidth=1.5)
        x_nodes = nodes_func(n)
        y_nodes = f(x_nodes)
        ax.plot(x_nodes, y_nodes, 'bo', markersize=4, label='узлы')
        ax.set_title(f'n = {n}')
        ax.set_xlabel('x')
        ax.set_ylabel('y')
        ax.legend()
        ax.grid(True)
        filename = f"{f_name}_{nodes_type}_n{n}.txt"
        save_points(filename, x_grid, y_interp)
    plt.tight_layout()
    plt.savefig(f"{f_name}_{nodes_type}.png")
    plt.close()

def plot_convergence(f, f_name):
    """График сходимости для равноотстоящих и чебышевских узлов"""
    n_values = np.arange(1, 62, 5)
    err_eq = []
    err_cheb = []
    x_grid = np.linspace(a,b,501)
    for n in n_values:
        # равноотстоящие
        x_nodes_eq = equidistant_nodes(n)
        y_nodes_eq = f(x_nodes_eq)
        dd_eq = divided_differences(x_nodes_eq, y_nodes_eq)
        y_interp_eq = np.array([newton_eval(x_nodes_eq, dd_eq, xi) for xi in x_grid])
        err_eq.append(np.max(np.abs(y_interp_eq - f(x_grid))))
        # чебышевские
        x_nodes_cheb = chebyshev_nodes(n)
        y_nodes_cheb = f(x_nodes_cheb)
        dd_cheb = divided_differences(x_nodes_cheb, y_nodes_cheb)
        y_interp_cheb = np.array([newton_eval(x_nodes_cheb, dd_cheb, xi) for xi in x_grid])
        err_cheb.append(np.max(np.abs(y_interp_cheb - f(x_grid))))
    plt.figure(figsize=(10,6))
    plt.semilogy(n_values, err_eq, 'b-o', label='равноотстоящие')
    plt.semilogy(n_values, err_cheb, 'r-s', label='чебышевские')
    plt.xlabel('Степень n')
    plt.ylabel('||r_n||')
    plt.title(f'Сходимость для {f_name}')
    plt.legend()
    plt.grid(True, which='both')
    plt.savefig(f"convergence_{f_name}.png")
    plt.close()

# Основная часть
if __name__ == "__main__":
    os.makedirs("points", exist_ok=True)
    os.chdir("points")
    
    functions = [(f1, "f1"), (f2, "f2")]
    node_types = [("равноотстоящие", equidistant_nodes), ("чебышевские", chebyshev_nodes)]
    
    for f, f_name in functions:
        for node_name, node_func in node_types:
            plot_interpolation(f, f_name, node_name, node_func)
    
    for f, f_name in functions:
        plot_convergence(f, f_name)
    
    print("Все графики построены и сохранены.")

## 2. Для функции с изломом \( f_2 \) наблюдается качественное различие:

- Чебышевские узлы существенно подавляют осцилляции; ошибка сначала убывает, а затем выходит на плато, ограниченное гладкостью функции.

## 3. Общий вывод: выбор узлов интерполяции критически влияет на сходимость. Для функций с ограниченной гладкостью (или разрывах производных) предпочтительнее использовать чебышевские узлы, которые минимизируют максимальную погрешность. Для аналитических функций оба подхода работоспособны, но чебышевские узлы дают более быструю сходимость.  

Таким образом, интерполяционный процесс по чебышевским узлам является более устойчивым и универсальным, особенно при интерполяции функций с особенностями.
